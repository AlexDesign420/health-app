# health-app

SwiftUI iOS app for reading selected Apple Health data and turning the latest daily values into simple health insights.

## Open in Xcode

1. Clone the repository on macOS.
2. Open `HealthAdvisor.xcodeproj` in Xcode.
3. Select the `HealthAdvisor` scheme and a physical iPhone target.
4. In Signing & Capabilities, select your Apple Developer team.
5. Run the app and grant Health access when iOS asks.

HealthKit data is only available on a real device. The simulator can build the app, but it will not provide real Apple Health values.
